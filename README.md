# Basic Website

A simple and responsive website built using HTML and CSS. This project includes two pages: **Home** and **Contact**, demonstrating fundamental front-end web development concepts, clean design, and easy navigation.

## Features
- Home Page
- Contact Page
- Responsive Layout
- Simple Navigation
- Clean and User-Friendly Interface

## Technologies Used
- HTML5
- CSS3

## Project Structure
```text
project-folder/
├── index.html      # Home Page
├── contact.html    # Contact Page
└── style.css       # Stylesheet
```

## Purpose
This project was created to practice and showcase basic web development skills, including webpage structure, styling, navigation, and responsive design using HTML and CSS.
